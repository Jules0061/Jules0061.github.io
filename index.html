FoliaKillstreaks

FoliaKillstreaks is one of my favorite plugins I've worked on and is currently used on CloneDupe. It's a highly optimized killstreak plugin with a lot of customizable features, including custom milestones, sounds, titles, chat broadcasts, and leaderboards, making it easy to configure for different types of servers. It's also fully open source, so anyone can view the code, contribute, or use it as a reference. You can check it out on GitHub at https://github.com/Jules0061/FoliaKillstreaks/.

KillStreakSpigot/Paper/FoliaCloneDupe

FoliaDupe

FoliaDupe is another one of my favorite plugins I've developed and is also currently used on CloneDupe. It's a highly optimized, open-source dupe plugin with a lot of customizable features, including configurable dupe amounts, drop overflow handling, customizable permissions, blacklisted materials, blacklisted NBT keywords, and fully customizable messages. You can check it out on GitHub at https://github.com/Jules0061/FoliaDupe/.
